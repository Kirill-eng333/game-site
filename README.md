Game Loot Landing Page
Современный и адаптивный лендинг, разработанный для игровой платформы. Проект представляет собой многосекционную посадочную страницу, ориентированную на геймеров, с акцентом на получение внутриигровых бонусов («лута») и знакомство с историей компании.

Основные разделы страницы
Header с баннером: Включает функциональную строку поиска, бургер-меню и главный экран с призывом к регистрации для получения игровых наград.

История и миссия: Информационный блок с карточками, описывающий путь компании с 1997 года и её основные ценности.

Игровой слайдер: Секция, демонстрирующая поддержку более 50 онлайн-игр с интерактивными элементами.

Игровые гаджеты: Презентационный блок с описанием характеристик устройств и функциональными карточками особенностей.

Интерактивный футер: Массивное визуальное завершение с полноэкранным изображением и призывом играть в бесплатные игры каждый месяц.

Технологии
HTML5: Семантическая верстка (использование тегов header, section, footer).

CSS3:

Flexbox: Использован для создания гибких сеток и выравнивания элементов в карточках и навигации.

Абсолютное позиционирование: Применяется для наложения текста поверх баннеров и футера.

Адаптивность: Использование object-fit: cover для корректного отображения изображений на разных экранах.
